# Deep Work Pro
A minimalist, distraction-free focus timer designed for students and athletes to track deep work sessions.

### Features
* **Focus Timer:** Customizable deep work blocks.
* **Heatmap:** Visualize your consistency over the last 28 days.
* **PB Tracker:** Track your personal best focus sessions.
* **Tagging:** Toggle between STUDY and VOLLEYBALL sessions.

### How to use
Built using HTML, CSS (Tailwind), and JavaScript. Hosted via Netlify.
